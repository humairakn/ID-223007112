
# Breast Cancer Classification

## Steps
- Encoding diagnosis (M→1, B→0)
- Dropped columns: id, Unnamed: 32
- MinMax Scaling
- Train/test split (80/20)
- KMeans clustering (k=2)
- KNN classifier (k=5)
